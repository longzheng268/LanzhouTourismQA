# 快速开始

## 🚀 3步启动

### 第1步：配置API密钥（必须）
复制 `config.json.example` 为 `config.json`，填入您的MIMO API密钥。

### 第2步：构建并运行（推荐）
```bash
./gradlew clean build run
```

### 第3步：或者仅运行（开发模式）
```bash
./gradlew run
```

## 🎯 VS Code调试

### 方法1：按F5直接调试
1. 打开VS Code
2. 按 `F5`
3. 选择配置：
   - **Debug LanzhouTourismQA** - 直接调试
   - **Run LanzhouTourismQA (Gradle)** - 先构建再运行

### 方法2：使用任务
1. 按 `Ctrl+Shift+P`
2. 输入 `Tasks: Run Task`
3. 选择：
   - `gradleBuild` - 构建
   - `gradleRun` - 构建并运行
   - `gradleClean` - 清理

## 🗄️ 数据库模式

### 启用数据库
编辑 `src/main/resources/config.json`：
```json
{
  "database": {
    "enabled": true,
    "host": "localhost",
    "port": 3306,
    "database": "qa_database",
    "username": "root",
    "password": "your_password"
  }
}
```

### 运行
```bash
./gradlew run
```

### 自动功能
- ✅ 连接MariaDB
- ✅ 加载 `qa_pairs` 表
- ✅ 保存聊天历史到 `chat_history` 表
- ✅ 数据库失败时自动回退到JSON模式

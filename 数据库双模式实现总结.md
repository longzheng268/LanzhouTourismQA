## 🎯 核心特性

### 双模式架构
```
QAService
├── 数据库模式 (enabled=true)
│   ├── 连接MariaDB
│   ├── 加载qa_pairs表
│   └── 保存到chat_history
│
└── JSON模式 (enabled=false)
    └── 加载knowledge_base.json
```

### 自动回退机制
```
启动 → 检查enabled
  ↓
true → 尝试连接数据库
  ↓
成功 → 使用数据库
失败 → 回退到JSON + 显示警告
```

### 聊天历史流程
```
用户提问 → RAG检索 → MIMO API → 保存到数据库
                                      ↓
                                chat_history表
```

CREATE TABLE qa_pairs (

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

## 🚀 使用方式

### 默认：JSON模式
无需任何配置，开箱即用：
```bash
.\\build.ps1
.\\run.ps1
```

### 可选：数据库模式
1. 修改 `config.json`：
```json
"database": { "enabled": true, "host": "localhost", "username": "root", "password": "your_password" }
```

2. 执行SQL脚本（C++项目中的setup_database_fixed.sql）

3. 运行程序
